# LinkedIn Scheduler V4 ??

Desktop LinkedIn post scheduler built with Python.

## Features

Schedule LinkedIn posts
Upload images
Calendar date picker
Exact time picker
Auto post using Chrome
Queue system
Delete scheduled posts
Live logs panel

## Installation

pip install -r requirements.txt
playwright install

## First Login Setup

python login_setup.py

## Start App

python main.py

Use responsibly and follow LinkedIn platform rules.
